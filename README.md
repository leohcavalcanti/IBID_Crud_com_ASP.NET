# IBIDEvents - Criando uma REST APIs com ASP.NET Core
Criação de um projeto de eventos.

## Tecnologias e ferramentas utilizadas
- Visual Studio 2022
- ASP.NET Core 7
- EF Core
- Swagger
- AutoMapper

## Funcionalidades
- Cadastro, Listagem, Detalhes, Atualização, e Remoção de Evento
- Cadastro de palestrantes
